1. SYSTEM OVERVIEW
================================================================================

The gesture detection system consists of 4 core components:

  1. GestureDetector (core/gesture_detector.py)
     - Detects hand gestures from MediaPipe landmarks
     - Supports 4 gestures: open_palm, fist, thumbs_up, two_open_palms
     - Built-in cooldown to prevent multiple triggers

  2. SentenceBuilder (core/sentence_builder.py)
     - Accumulates signs/letters into words and sentences
     - Supports add_letter(), add_space(), backspace(), clear(), speak()

  3. Gesture Display (utils/gesture_display.py)
     - Renders gesture feedback on video frame
     - Shows sentence being built

  4. Hand Detector (core/hand_detector.py)
     - Uses MediaPipe to detect hand landmarks
     - Returns normalized coordinates (0.0-1.0)

3. GESTURE TYPES AND HOW THEY WORK
================================================================================

GESTURE #1: Open Palm (Returns: 'space')
   Hand Position: All 5 fingers extended upward
   Detection Logic:
     - At least 4 fingertips must be above the hand center (y < center_y)
     - Horizontal spread > 0.1 (normalized width)
     - All fingertips facing camera (z-depth consistent < 0.1)
   Action: Adds space to current sentence

GESTURE #2: Fist (Returns: 'backspace')
   Hand Position: All fingers curled tightly
   Detection Logic:
     - All fingertips within 0.15 normalized distance from wrist
     - Fingers pulled in tight (curl detector)
   Action: Backspace - removes last character

GESTURE #3: Thumbs Up (Returns: 'speak')
   Hand Position: Thumb extended upward, other fingers curled
   Detection Logic:
     - Thumb tip is 0.3+ normalized distance above wrist
     - Other 4 fingers curled (distance < 0.15)
   Action: Speak - reads sentence aloud via text-to-speech

GESTURE #4: Two Open Palms (Returns: 'clear')
   Hand Position: Both hands open with palms facing camera
   Detection Logic:
     - Exactly 2 hands detected
     - Both hands pass is_open_palm() check
   Action: Clear - resets sentence to empty string

================================================================================
4. GESTURE DETECTION ALGORITHMS (TECHNICAL DETAILS)
================================================================================

The GestureDetector class uses MediaPipe hand landmark indices:

  Landmark Index Map:
    0: Wrist (base of hand)
    1-4: Thumb (base → tip)
    5-8: Index finger (base → tip)
    9-12: Middle finger (base → tip)
    13-16: Ring finger (base → tip)
    17-20: Pinky finger (base → tip)

Distance Thresholds (all normalized to 0.0-1.0 coordinate space):
  - spread_threshold = 0.1  (minimum horizontal spread for open palm)
  - curl_threshold = 0.15   (maximum distance for curled finger)
  - thumbs_threshold = 0.3  (minimum vertical distance for thumbs up)

Cooldown Mechanism:
  - 500ms (0.5 seconds) between gesture triggers
  - Prevents accidental double-triggers
  - Reset automatically when cooldown expires

================================================================================
5. USING THE SENTENCEBUILDER
================================================================================

The SentenceBuilder class manages text being built by the user:

  Methods:

    add_letter(letter: str)
    - Adds a single letter or sign to current word
    - Usage: sentence_builder.add_letter('A')

    add_space()
    - Completes current word and starts new one
    - Usage: sentence_builder.add_space()

    backspace()
    - Removes last character from current word
    - If word is empty, removes last word
    - Usage: sentence_builder.backspace()

    clear()
    - Resets everything to empty string
    - Usage: sentence_builder.clear()

    speak()
    - Reads current sentence aloud (requires text-to-speech)
    - Usage: sentence_builder.speak()
    - NOTE: Requires pyttsx3 or similar TTS library

    current_sentence() → str
    - Returns the current full sentence being built
    - Usage: text = sentence_builder.current_sentence()

================================================================================
6. VISUAL FEEDBACK SYSTEM
================================================================================

Two drawing functions provide visual feedback:

Function 1: draw_gesture_feedback(frame, gesture_name)
  Input:
    - frame: OpenCV image (BGR format)
    - gesture_name: String like 'space', 'backspace', 'speak', 'clear' or None
  Output:
    - Modified frame with gesture feedback text
  Behavior:
    - If gesture_name is valid, displays colored text on frame (top-left)
    - If gesture_name is None, returns frame unchanged
    - Colors: Green text for valid gestures

Function 2: draw_sentence_display(frame, sentence)
  Input:
    - frame: OpenCV image (BGR format)
    - sentence: String with current sentence text
  Output:
    - Modified frame with sentence display
  Behavior:
    - If sentence is empty/None, returns frame unchanged (no black box!)
    - If sentence has text, displays it with background rectangle (bottom area)
    - Large, readable font for user visibility

8. MAIN.PY INTEGRATION EXAMPLE
================================================================================

Here's the recommended structure for your main application:

    import cv2
    from core.hand_detector import HandDetector
    from core.gesture_detector import GestureDetector
    from core.sentence_builder import SentenceBuilder
    from utils.gesture_display import draw_gesture_feedback, draw_sentence_display

    class ChironaApp:
        def __init__(self):
            self.hand_detector = HandDetector()
            self.gesture_detector = GestureDetector()
            self.sentence_builder = SentenceBuilder()
            
        def process_frame(self, frame):
            # Step 1: Detect hands in frame
            hands = self.hand_detector.detect(frame)
            
            # Step 2: Recognize gesture from hand positions
            gesture = self.gesture_detector.detect_gesture(hands)
            
            # Step 3: Process gesture to build sentence
            if gesture == 'space':
                self.sentence_builder.add_space()
            elif gesture == 'backspace':
                self.sentence_builder.backspace()
            elif gesture == 'speak':
                self.sentence_builder.speak()
            elif gesture == 'clear':
                self.sentence_builder.clear()
            
            # Step 4: Draw feedback on frame
            frame = draw_gesture_feedback(frame, gesture)
            sentence = self.sentence_builder.current_sentence()
            frame = draw_sentence_display(frame, sentence)
            
            return frame

        def run(self):
            cap = cv2.VideoCapture(0)
            while True:
                ret, frame = cap.read()
                if not ret:
                    break
                
                frame = self.process_frame(frame)
                cv2.imshow('Chirona', frame)
                
                if cv2.waitKey(1) & 0xFF == ord('q'):
                    break
            
            cap.release()
            cv2.destroyAllWindows()

    if __name__ == '__main__':
        app = ChironaApp()
        app.run()

10. TROUBLESHOOTING
================================================================================

Issue: Gestures not detected
Solution:
  - Check that hand_detector is properly detecting hands
  - Verify hand position matches gesture definition
  - Confirm gesture_detector.detect_gesture() is being called with hands
  - Check cooldown hasn't blocked detection (wait 0.5 seconds)

Issue: No visual feedback appearing
Solution:
  - Verify draw_gesture_feedback() is being called after detect_gesture()
  - Check that gesture string is not None
  - Confirm frame is being displayed with cv2.imshow()

Issue: Sentence not appearing
Solution:
  - Verify sentence_builder.current_sentence() returns text
  - Check draw_sentence_display() is being called with non-empty sentence
  - Confirm add_space(), add_letter(), etc. are being called

Issue: Text-to-speech not working
Solution:
  - Install pyttsx3: pip install pyttsx3
  - Verify windows.h and required audio dependencies are installed
  - Check sentence_builder.speak() implementation

11. PERFORMANCE CONSIDERATIONS
================================================================================

- GestureDetector runs in O(1) time (fixed 21 landmarks)
- SentenceBuilder is O(n) where n = length of current sentence
- Visual drawing is fast using OpenCV
- Gesture detection adds ~5-10ms overhead per frame
- Cooldown timing is important: 500ms prevents accidental double-triggers

To optimize:
  1. Run gesture detection at full frame rate (30-60 fps)
  2. Keep video resolution at 640x480 or lower for hand detection
  3. Cache hand detector results if processing multiple gestures per frame
  4. Use threading for text-to-speech to avoid frame delays

    def __init__(self):
        self.landmark_threshold_spread = 0.1      # Lower = stricter open palm
        self.landmark_threshold_curl = 0.15       # Lower = stricter fist
        self.landmark_threshold_thumbs = 0.3      # Lower = stricter thumbs up
        self.cooldown_seconds = 0.5               # Increase for more safety

And in main.py, adjust OPTION A threshold:

    if unique_predictions >= 2:  # Lower = more lenient, easily triggered
                                  # Higher = more strict, harder to trigger

Recommended values:
  - 2: Very easy to trigger (current)
  - 3: Medium difficulty
  - 4+: Very strict, requires very steady hand position
