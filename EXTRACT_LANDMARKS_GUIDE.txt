STEP 2: Define Helper Functions
================================================================================

2a. GET HAND BOUNDING BOX
--------------------------
Function: get_hand_bbox(landmarks)
Purpose: Calculate the bounding box containing all 21 hand landmarks

Logic:
  1. Extract x coordinates from all 21 landmarks → xs = [lm.x for lm in ...]
  2. Extract y coordinates from all 21 landmarks → ys = [lm.y for lm in ...]
  3. Find min/max for both: x_min, y_min, x_max, y_max
  4. Return tuple: (x_min, y_min, x_max, y_max)

Input:  MediaPipe landmarks object
Output: tuple of 4 normalized coordinates [0, 1]


2b. NORMALIZE LANDMARKS
------------------------
Function: normalize_landmarks(landmarks, bbox)
Purpose: Convert 21 landmarks to 42 values relative to bounding box
         (Makes them position-invariant)

Logic:
  1. Unpack bbox: x_min, y_min, x_max, y_max = bbox
  2. Calculate width = x_max - x_min (prevent division by zero!)
  3. Calculate height = y_max - y_min (prevent division by zero!)
  4. For each of 21 landmarks:
     - Normalize x: (landmark.x - x_min) / width
     - Normalize y: (landmark.y - y_min) / height
     - Append BOTH to output list (creates 42 values total)
  5. Return list of 42 normalized floats

Input:  MediaPipe landmarks object, bbox tuple
Output: list of 42 floats (21 x-y pairs), each in range [0, 1]

================================================================================
KEY IMPLEMENTATION DETAILS
================================================================================

STATIC MODE:
  - Use static_image_mode=True when initializing detector for batch processing
  - Your main.py uses False for video (continuous frame tracking)
  - Batch processing doesn't need temporal continuity

NORMALIZATION:
  - Normalize relative to hand's BOUNDING BOX, not image bounds
  - This makes landmarks position-invariant (works regardless of hand position)
  - Also makes them scale-invariant (works regardless of hand size)
  - Output range: [0, 1] for all 42 values

SKIP STRATEGY:
  - Check: if results.multi_hand_landmarks:
  - Log filename + class of skipped images
  - Don't process images with no detections
  - Keep track for final report

ARRAY SHAPE & TYPES:
  - Final all_data.shape = (N_samples, 42)
  - Final all_labels.shape = (N_samples,)
  - Use np.float32 for data (ML libraries expect this)
  - Use np.int32 for labels

DATA STRUCTURE IN PICKLE:
  File contains single dict with two keys:
  {
      'data': np.array of shape (N, 42),     # All landmarks, all samples
      'labels': np.array of shape (N,),       # Class ID for each sample
  }
  
  To load later:
      with open('data/landmarks.pickle', 'rb') as f:
          dataset = pickle.load(f)
      X = dataset['data']       # Training features
      y = dataset['labels']     # Training labels
